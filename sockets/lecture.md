### **–û—Å–Ω–æ–≤–∏ —Ä–æ–±–æ—Ç–∏ Socket**

–°–æ–∫–µ—Ç–∏ ‚Äì —Ü–µ –ø—Ä–æ–≥—Ä–∞–º–Ω–∏–π —ñ–Ω—Ç–µ—Ä—Ñ–µ–π—Å –¥–ª—è –∫–æ–º—É–Ω—ñ–∫–∞—Ü—ñ—ó –º—ñ–∂ –ø—Ä–∏—Å—Ç—Ä–æ—è–º–∏ —É –º–µ—Ä–µ–∂—ñ. –í–æ–Ω–∏ –¥–æ–∑–≤–æ–ª—è—é—Ç—å –¥–≤–æ–º –ø—Ä–æ—Ü–µ—Å–∞–º, —è–∫—ñ –º–æ–∂—É—Ç—å –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ —è–∫ –Ω–∞ –æ–¥–Ω–æ–º—É –∫–æ–º–ø'—é—Ç–µ—Ä—ñ, —Ç–∞–∫ —ñ –Ω–∞ —Ä—ñ–∑–Ω–∏—Ö –ø—Ä–∏—Å—Ç—Ä–æ—è—Ö, –æ–±–º—ñ–Ω—é–≤–∞—Ç–∏—Å—è –¥–∞–Ω–∏–º–∏ —á–µ—Ä–µ–∑ –º–µ—Ä–µ–∂—É. –í –æ—Å–Ω–æ–≤—ñ —Å–æ–∫–µ—Ç–Ω–æ–≥–æ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è –ª–µ–∂–∏—Ç—å –º–æ–¥–µ–ª—å –∫–ª—ñ—î–Ω—Ç-—Å–µ—Ä–≤–µ—Ä, –¥–µ –æ–¥–∏–Ω –ø—Ä–æ—Ü–µ—Å –≤–∏—Å—Ç—É–ø–∞—î —è–∫ —Å–µ—Ä–≤–µ—Ä, —â–æ –æ—á—ñ–∫—É—î –Ω–∞ –∑'—î–¥–Ω–∞–Ω–Ω—è, –∞ —ñ–Ω—à–∏–π ‚Äì —è–∫ –∫–ª—ñ—î–Ω—Ç, —è–∫–∏–π —ñ–Ω—ñ—Ü—ñ—é—î —Ü–µ –∑'—î–¥–Ω–∞–Ω–Ω—è.

–°–æ–∫–µ—Ç–∏ –ø—Ä–∞—Ü—é—é—Ç—å –ø–æ–≤–µ—Ä—Ö –ø—Ä–æ—Ç–æ–∫–æ–ª—ñ–≤ –ø–µ—Ä–µ–¥–∞—á—ñ –¥–∞–Ω–∏—Ö. –ù–∞–π–ø–æ—à–∏—Ä–µ–Ω—ñ—à–∏–º–∏ —î TCP (Transmission Control Protocol) —ñ UDP (User Datagram Protocol). TCP –∑–∞–±–µ–∑–ø–µ—á—É—î –Ω–∞–¥—ñ–π–Ω—É, –≤–ø–æ—Ä—è–¥–∫–æ–≤–∞–Ω—É —Ç–∞ –±–µ–∑–ø–æ–º–∏–ª–∫–æ–≤—É –¥–æ—Å—Ç–∞–≤–∫—É –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω—å, –≤—Å—Ç–∞–Ω–æ–≤–ª—é—é—á–∏ —Å—Ç—ñ–π–∫–µ –∑'—î–¥–Ω–∞–Ω–Ω—è –º—ñ–∂ –∫–ª—ñ—î–Ω—Ç–æ–º —ñ —Å–µ—Ä–≤–µ—Ä–æ–º. –¶–µ–π –ø—Ä–æ—Ç–æ–∫–æ–ª –ø—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –∑–∞—Å—Ç–æ—Å—É–Ω–∫—ñ–≤, –¥–µ –≤–∞–∂–ª–∏–≤–∞ —Ü—ñ–ª—ñ—Å–Ω—ñ—Å—Ç—å –¥–∞–Ω–∏—Ö, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –¥–ª—è —á–∞—Ç—ñ–≤, —Ñ–∞–π–ª–æ–æ–±–º—ñ–Ω—É —á–∏ –≤–µ–±-–∑–∞–ø–∏—Ç—ñ–≤. –ù–∞—Ç–æ–º—ñ—Å—Ç—å UDP —î –±—ñ–ª—å—à –ª–µ–≥–∫–∏–º —ñ —à–≤–∏–¥–∫–∏–º, –∞–ª–µ –Ω–µ –≥–∞—Ä–∞–Ω—Ç—É—î –¥–æ—Å—Ç–∞–≤–∫–∏ —á–∏ –ø—Ä–∞–≤–∏–ª—å–Ω–æ–≥–æ –ø–æ—Ä—è–¥–∫—É –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –ø–∞–∫–µ—Ç—ñ–≤, —â–æ —Ä–æ–±–∏—Ç—å –π–æ–≥–æ —ñ–¥–µ–∞–ª—å–Ω–∏–º –¥–ª—è –∑–∞—Å—Ç–æ—Å—É–Ω–∫—ñ–≤, –¥–µ —à–≤–∏–¥–∫—ñ—Å—Ç—å –≤–∞–∂–ª–∏–≤—ñ—à–∞ –∑–∞ —Ç–æ—á–Ω—ñ—Å—Ç—å, —Ç–∞–∫–∏—Ö —è–∫ –≤—ñ–¥–µ–æ—Å—Ç—Ä—ñ–º–∏ –∞–±–æ –æ–Ω–ª–∞–π–Ω-—ñ–≥—Ä–∏.

–°–æ–∫–µ—Ç–Ω–∞ –≤–∑–∞—î–º–æ–¥—ñ—è –ø–æ—á–∏–Ω–∞—î—Ç—å—Å—è –∑ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑–∞—Ü—ñ—ó —Å–æ–∫–µ—Ç–∞, –ø—ñ—Å–ª—è —á–æ–≥–æ —Å–µ—Ä–≤–µ—Ä –ø—Ä–∏–≤‚Äô—è–∑—É—î –π–æ–≥–æ –¥–æ –ø–µ–≤–Ω–æ—ó IP-–∞–¥—Ä–µ—Å–∏ —Ç–∞ –ø–æ—Ä—Ç—É, —â–æ –¥–æ–∑–≤–æ–ª—è—î –∫–ª—ñ—î–Ω—Ç–∞–º –∑–Ω–∞–π—Ç–∏ –π–æ–≥–æ —É –º–µ—Ä–µ–∂—ñ. –°–µ—Ä–≤–µ—Ä –ø–µ—Ä–µ–≤–æ–¥–∏—Ç—å —Å–æ–∫–µ—Ç —É —Ä–µ–∂–∏–º –æ—á—ñ–∫—É–≤–∞–Ω–Ω—è –ø—ñ–¥–∫–ª—é—á–µ–Ω—å, —ñ –∫–æ–ª–∏ –∫–ª—ñ—î–Ω—Ç —ñ–Ω—ñ—Ü—ñ—é—î –∑'—î–¥–Ω–∞–Ω–Ω—è, —Å–µ—Ä–≤–µ—Ä –ø—Ä–∏–π–º–∞—î –π–æ–≥–æ —Ç–∞ —Å—Ç–≤–æ—Ä—é—î –Ω–æ–≤–∏–π —Å–æ–∫–µ—Ç –¥–ª—è –æ–±–º—ñ–Ω—É –¥–∞–Ω–∏–º–∏ –∑ —Ü–∏–º –∫–æ–Ω–∫—Ä–µ—Ç–Ω–∏–º –∫–ª—ñ—î–Ω—Ç–æ–º. –ü—ñ—Å–ª—è –≤—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–Ω—è –∑'—î–¥–Ω–∞–Ω–Ω—è –æ–±–∏–¥–≤—ñ —Å—Ç–æ—Ä–æ–Ω–∏ –º–æ–∂—É—Ç—å –Ω–∞–¥—Å–∏–ª–∞—Ç–∏ —Ç–∞ –æ—Ç—Ä–∏–º—É–≤–∞—Ç–∏ –¥–∞–Ω—ñ, –ø–æ–∫–∏ –Ω–µ –∑–∞–∫—Ä–∏—é—Ç—å —Å–æ–∫–µ—Ç.

–£ Python —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∞ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∞ `socket` –¥–æ–∑–≤–æ–ª—è—î –ø—Ä–∞—Ü—é–≤–∞—Ç–∏ –∑ –º–µ—Ä–µ–∂–µ–≤–∏–º–∏ —Å–æ–∫–µ—Ç–∞–º–∏ –Ω–∞ –Ω–∏–∑—å–∫–æ–º—É —Ä—ñ–≤–Ω—ñ. –©–æ–± —Å—Ç–≤–æ—Ä–∏—Ç–∏ TCP-—Å–æ–∫–µ—Ç, –¥–æ—Å—Ç–∞—Ç–Ω—å–æ –≤–∏–∫–ª–∏–∫–∞—Ç–∏ `socket.socket(socket.AF_INET, socket.SOCK_STREAM)`, –¥–µ `AF_INET` –≤–∫–∞–∑—É—î –Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è IPv4, –∞ `SOCK_STREAM` –æ–∑–Ω–∞—á–∞—î —Ä–æ–±–æ—Ç—É –∑ TCP. –î–ª—è UDP –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è `SOCK_DGRAM`. –ü—ñ—Å–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è —Å–æ–∫–µ—Ç–∞ –π–æ–≥–æ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –ø—Ä–∏–≤‚Äô—è–∑–∞—Ç–∏ –¥–æ –ø–µ–≤–Ω–æ–≥–æ –ø–æ—Ä—Ç—É, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ `bind()`, –ø—ñ—Å–ª—è —á–æ–≥–æ —Å–µ—Ä–≤–µ—Ä –º–æ–∂–µ –ø–æ—á–∞—Ç–∏ —Å–ª—É—Ö–∞—Ç–∏ –≤—Ö—ñ–¥–Ω—ñ –∑'—î–¥–Ω–∞–Ω–Ω—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é `listen()`. –ö–æ–ª–∏ –∫–ª—ñ—î–Ω—Ç –ø—ñ–¥–∫–ª—é—á–∞—î—Ç—å—Å—è, —Å–µ—Ä–≤–µ—Ä –ø—Ä–∏–π–º–∞—î –∑'—î–¥–Ω–∞–Ω–Ω—è –≤–∏–∫–ª–∏–∫–æ–º `accept()`, –ø—ñ—Å–ª—è —á–æ–≥–æ –ø–æ—á–∏–Ω–∞—î—Ç—å—Å—è –æ–±–º—ñ–Ω –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è–º–∏ —á–µ—Ä–µ–∑ `send()` —ñ `recv()`.

–ö–ª—ñ—î–Ω—Ç, —É —Å–≤–æ—é —á–µ—Ä–≥—É, —Å—Ç–≤–æ—Ä—é—î —Å–æ–∫–µ—Ç —ñ –≤—Å—Ç–∞–Ω–æ–≤–ª—é—î –∑'—î–¥–Ω–∞–Ω–Ω—è, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ `connect()`, –ø—ñ—Å–ª—è —á–æ–≥–æ –º–æ–∂–µ –Ω–∞–¥—Å–∏–ª–∞—Ç–∏ —Ç–∞ –æ—Ç—Ä–∏–º—É–≤–∞—Ç–∏ –¥–∞–Ω—ñ. –û—Å–Ω–æ–≤–Ω–æ—é –≤—ñ–¥–º—ñ–Ω–Ω—ñ—Å—Ç—é –∫–ª—ñ—î–Ω—Ç—Å—å–∫–æ—ó —á–∞—Å—Ç–∏–Ω–∏ —î –≤—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å –æ–ø–µ—Ä–∞—Ü—ñ–π `bind()` —ñ `listen()`, –æ—Å–∫—ñ–ª—å–∫–∏ –∫–ª—ñ—î–Ω—Ç –Ω–µ –æ—á—ñ–∫—É—î –ø—ñ–¥–∫–ª—é—á–µ–Ω—å, –∞ –ª–∏—à–µ —ñ–Ω—ñ—Ü—ñ—é—î —ó—Ö.

–†–æ–∑—É–º—ñ–Ω–Ω—è —Ü—ñ—î—ó –º–æ–¥–µ–ª—ñ —î –æ—Å–Ω–æ–≤–æ—é –¥–ª—è –ø–æ–¥–∞–ª—å—à–æ–≥–æ –≤–∏–≤—á–µ–Ω–Ω—è –ø—Ä–∞–∫—Ç–∏—á–Ω–∏—Ö –ø—Ä–∏–∫–ª–∞–¥—ñ–≤, –∑–æ–∫—Ä–µ–º–∞ –ø–æ–±—É–¥–æ–≤–∏ –µ—Ö–æ-—Å–µ—Ä–≤–µ—Ä–∞ —Ç–∞ —á–∞—Ç-—Å–µ—Ä–≤–µ—Ä–∞, –∞ —Ç–∞–∫–æ–∂ –ø–µ—Ä–µ—Ö–æ–¥—É –¥–æ WebSockets, —è–∫—ñ —Ä–æ–∑—à–∏—Ä—é—é—Ç—å –º–æ–∂–ª–∏–≤–æ—Å—Ç—ñ –ø–æ—Å—Ç—ñ–π–Ω–æ–≥–æ –∑'—î–¥–Ω–∞–Ω–Ω—è –º—ñ–∂ –∫–ª—ñ—î–Ω—Ç–æ–º —ñ —Å–µ—Ä–≤–µ—Ä–æ–º.

### **–ê—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∏ –∫–æ–º—É–Ω—ñ–∫–∞—Ü—ñ—ó —É –º–µ—Ä–µ–∂–∞—Ö**

–ü–µ—Ä–µ–¥ —Ç–∏–º —è–∫ –∑–∞–≥–ª–∏–±–ª—é–≤–∞—Ç–∏—Å—è —É WebSockets, –≤–∞—Ä—Ç–æ –∑—Ä–æ–∑—É–º—ñ—Ç–∏ –æ—Å–Ω–æ–≤–Ω—ñ –º–æ–¥–µ–ª—ñ –º–µ—Ä–µ–∂–µ–≤–æ—ó –≤–∑–∞—î–º–æ–¥—ñ—ó. –ö–ª–∞—Å–∏—á–Ω–∞ –º–æ–¥–µ–ª—å –∫–ª—ñ—î–Ω—Ç-—Å–µ—Ä–≤–µ—Ä –ø–µ—Ä–µ–¥–±–∞—á–∞—î, —â–æ –æ–¥–∏–Ω —Å–µ—Ä–≤–µ—Ä –æ–±—Ä–æ–±–ª—è—î –∑–∞–ø–∏—Ç–∏ –≤—ñ–¥ –∫–ª—ñ—î–Ω—Ç—ñ–≤, —Ç–æ–¥—ñ —è–∫ —É Peer-to-Peer (P2P) –º–æ–¥–µ–ª—ñ –∫–æ–∂–µ–Ω –≤—É–∑–æ–ª –º–æ–∂–µ –±—É—Ç–∏ —ñ –∫–ª—ñ—î–Ω—Ç–æ–º, —ñ —Å–µ—Ä–≤–µ—Ä–æ–º –æ–¥–Ω–æ—á–∞—Å–Ω–æ. WebSockets –Ω–∞–π—á–∞—Å—Ç—ñ—à–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è —É –∫–ª—ñ—î–Ω—Ç-—Å–µ—Ä–≤–µ—Ä–Ω–∏—Ö –¥–æ–¥–∞—Ç–∫–∞—Ö, –∞–ª–µ —î —Å–ø–æ—Å–æ–±–∏ —ñ–Ω—Ç–µ–≥—Ä—É–≤–∞—Ç–∏ —ó—Ö —É P2P —á–µ—Ä–µ–∑ –ø—Ä–æ–∫—Å—ñ –∞–±–æ —Ä–µ—Ç—Ä–∞–Ω—Å–ª—è—Ç–æ—Ä–∏.

–¢–∞–∫–æ–∂ —Å–ª—ñ–¥ —Ä–æ–∑–≥–ª—è–Ω—É—Ç–∏ –ø—ñ–¥—Ö–æ–¥–∏ **Push vs. Pull**. HTTP-–∑–∞–ø–∏—Ç–∏ –ø—Ä–∞—Ü—é—é—Ç—å –∑–∞ pull-–º–æ–¥–µ–ª–ª—é (–∫–ª—ñ—î–Ω—Ç –∑–∞–ø–∏—Ç—É—î ‚Äî —Å–µ—Ä–≤–µ—Ä –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î), —Ç–æ–¥—ñ —è–∫ WebSockets –¥–æ–∑–≤–æ–ª—è—é—Ç—å —Å–µ—Ä–≤–µ—Ä—É "–ø—É—à–∏—Ç–∏" –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è –∫–ª—ñ—î–Ω—Ç—É –±–µ–∑ –æ—á—ñ–∫—É–≤–∞–Ω–Ω—è –∑–∞–ø–∏—Ç—ñ–≤. –¶–µ –≤–∞–∂–ª–∏–≤–æ –¥–ª—è —Ä–µ–∞–ª—å–Ω–æ–≥–æ —á–∞—Å—É, –¥–µ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –º–∏—Ç—Ç—î–≤–æ —Ä–µ–∞–≥—É–≤–∞—Ç–∏ –Ω–∞ –ø–æ–¥—ñ—ó.

### **Push vs. Pull: –Ø–∫ –ø—Ä–∞—Ü—é—î –ø–µ—Ä–µ–¥–∞—á–∞ –¥–∞–Ω–∏—Ö —É –≤–µ–±—ñ**

–£ –≤–µ–±—ñ —ñ—Å–Ω—É—é—Ç—å –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω—ñ –ø—ñ–¥—Ö–æ–¥–∏ –¥–ª—è –ø–µ—Ä–µ–¥–∞—á—ñ –¥–∞–Ω–∏—Ö –º—ñ–∂ –∫–ª—ñ—î–Ω—Ç–æ–º —ñ —Å–µ—Ä–≤–µ—Ä–æ–º: **Pull** (–≤–∏—Ç—è–≥—É–≤–∞–Ω–Ω—è) —Ç–∞ **Push** (–ø—Ä–æ—à—Ç–æ–≤—Ö—É–≤–∞–Ω–Ω—è). –í–æ–Ω–∏ –≤—ñ–¥—Ä—ñ–∑–Ω—è—é—Ç—å—Å—è —Ç–∏–º, **—Ö—Ç–æ —ñ–Ω—ñ—Ü—ñ—é—î –∫–æ–º—É–Ω—ñ–∫–∞—Ü—ñ—é** —Ç–∞ —è–∫ —á–∞—Å—Ç–æ –æ–Ω–æ–≤–ª—é—é—Ç—å—Å—è –¥–∞–Ω—ñ.

## **Pull (–ö–ª—ñ—î–Ω—Ç —ñ–Ω—ñ—Ü—ñ—é—î –∑–∞–ø–∏—Ç)**

Pull-–º–æ–¥–µ–ª—å –ø—Ä–∞—Ü—é—î –∑–∞ –ø—Ä–∏–Ω—Ü–∏–ø–æ–º **"–∑–∞–ø–∏—Ç-–≤—ñ–¥–ø–æ–≤—ñ–¥—å"**:

1. –ö–ª—ñ—î–Ω—Ç (–±—Ä–∞—É–∑–µ—Ä –∞–±–æ —ñ–Ω—à–∞ –ø—Ä–æ–≥—Ä–∞–º–∞) –Ω–∞–¥—Å–∏–ª–∞—î –∑–∞–ø–∏—Ç –Ω–∞ —Å–µ—Ä–≤–µ—Ä (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, —á–µ—Ä–µ–∑ HTTP).
2. –°–µ—Ä–≤–µ—Ä –æ—Ç—Ä–∏–º—É—î –∑–∞–ø–∏—Ç, –æ–±—Ä–æ–±–ª—è—î –π–æ–≥–æ —Ç–∞ –Ω–∞–¥—Å–∏–ª–∞—î –≤—ñ–¥–ø–æ–≤—ñ–¥—å.
3. –Ø–∫—â–æ –∫–ª—ñ—î–Ω—Ç—É –ø–æ—Ç—Ä—ñ–±–Ω—ñ –Ω–æ–≤—ñ –¥–∞–Ω—ñ, –≤—ñ–Ω –ø–æ–≤–∏–Ω–µ–Ω –∑—Ä–æ–±–∏—Ç–∏ —â–µ –æ–¥–∏–Ω –∑–∞–ø–∏—Ç.

–¶–µ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∏–π –º–µ—Ö–∞–Ω—ñ–∑–º –¥–ª—è –±—ñ–ª—å—à–æ—Å—Ç—ñ –≤–µ–±—Å–∞–π—Ç—ñ–≤. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, –∫–æ–ª–∏ –º–∏ –≤—ñ–¥–∫—Ä–∏–≤–∞—î–º–æ —Å—Ç–æ—Ä—ñ–Ω–∫—É –≤ –±—Ä–∞—É–∑–µ—Ä—ñ, –≤—ñ–Ω –∑–∞–ø–∏—Ç—É—î HTML-–∫–æ–¥, —Å–µ—Ä–≤–µ—Ä –π–æ–≥–æ –ø–æ–≤–µ—Ä—Ç–∞—î, —ñ –∫–æ–º—É–Ω—ñ–∫–∞—Ü—ñ—è –∑–∞–∫—ñ–Ω—á—É—î—Ç—å—Å—è.

### **–ü—Ä–∏–∫–ª–∞–¥–∏ Pull-–∫–æ–º—É–Ω—ñ–∫–∞—Ü—ñ—ó**

- **HTTP-–∑–∞–ø–∏—Ç–∏ (REST API, GraphQL)** ‚Äì –±—Ä–∞—É–∑–µ—Ä –∑–∞–ø–∏—Ç—É—î –¥–∞–Ω—ñ, –∞ —Å–µ—Ä–≤–µ—Ä –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î.
- **AJAX polling** ‚Äì –∫–ª—ñ—î–Ω—Ç –∫–æ–∂–Ω—ñ N —Å–µ–∫—É–Ω–¥ –Ω–∞–¥—Å–∏–ª–∞—î –∑–∞–ø–∏—Ç (`setInterval(fetchData, 5000)`).
- **Long polling** ‚Äì –∫–ª—ñ—î–Ω—Ç –Ω–∞–¥—Å–∏–ª–∞—î –∑–∞–ø–∏—Ç, –∞ —Å–µ—Ä–≤–µ—Ä —á–µ–∫–∞—î –ø–æ—è–≤–∏ –Ω–æ–≤–∏—Ö –¥–∞–Ω–∏—Ö –ø–µ—Ä–µ–¥ —Ç–∏–º, —è–∫ –≤—ñ–¥–ø–æ–≤—ñ—Å—Ç–∏ (—É—Ç—Ä–∏–º—É—î –∑'—î–¥–Ω–∞–Ω–Ω—è).

## **1. WebSockets (–î–≤–æ—Å—Ç–æ—Ä–æ–Ω–Ω—ñ–π –∑–≤'—è–∑–æ–∫, –ø–æ—Å—Ç—ñ–π–Ω–µ –∑'—î–¥–Ω–∞–Ω–Ω—è)**

### **–†–µ–∞–ª—ñ–∑–∞—Ü—ñ—è –ø—Ä–æ—Å—Ç–æ–≥–æ —á–∞—Ç-—Å–µ—Ä–≤–µ—Ä–∞ –Ω–∞ —Å–æ–∫–µ—Ç–∞—Ö**

–ï—Ö–æ-—Å–µ—Ä–≤–µ—Ä —î —Ö–æ—Ä–æ—à–æ—é –≤—ñ–¥–ø—Ä–∞–≤–Ω–æ—é —Ç–æ—á–∫–æ—é, –∞–ª–µ –¥–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è –ø–æ–≤–Ω–æ—Ü—ñ–Ω–Ω–æ–≥–æ —á–∞—Ç—É –ø–æ—Ç—Ä—ñ–±–Ω–æ, —â–æ–± —Å–µ—Ä–≤–µ—Ä –º—ñ–≥ –æ–±—Ä–æ–±–ª—è—Ç–∏ –∫—ñ–ª—å–∫–∞ –∫–ª—ñ—î–Ω—Ç—ñ–≤ –æ–¥–Ω–æ—á–∞—Å–Ω–æ. –Ø–∫—â–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –æ–¥–Ω–æ–ø–æ—Ç–æ–∫–æ–≤—É –º–æ–¥–µ–ª—å, –∫–æ–∂–µ–Ω –∫–ª—ñ—î–Ω—Ç –±–ª–æ–∫—É–≤–∞—Ç–∏–º–µ —Ä–æ–±–æ—Ç—É —Å–µ—Ä–≤–µ—Ä–∞, –ø–æ–∫–∏ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è –ø–µ—Ä–µ–¥–∞—á–∞ –¥–∞–Ω–∏—Ö. –©–æ–± —É–Ω–∏–∫–Ω—É—Ç–∏ —Ü—å–æ–≥–æ, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å –±–∞–≥–∞—Ç–æ–ø–æ—Ç–æ–∫–æ–≤—É –∞–±–æ –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—É –º–æ–¥–µ–ª—å. –£ —Ü—å–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ —Ä–µ–∞–ª—ñ–∑—É—î–º–æ —á–∞—Ç-—Å–µ—Ä–≤–µ—Ä –Ω–∞ –æ—Å–Ω–æ–≤—ñ –ø–æ—Ç–æ–∫—ñ–≤ (`threading`), —â–æ –¥–æ–∑–≤–æ–ª—è—î –æ–±—Ä–æ–±–ª—è—Ç–∏ –∫–æ–∂–Ω–æ–≥–æ –∫–ª—ñ—î–Ω—Ç–∞ –≤ –æ–∫—Ä–µ–º–æ–º—É –ø–æ—Ç–æ—Ü—ñ.

–û—Å–Ω–æ–≤–Ω–∞ —ñ–¥–µ—è –ø–æ–ª—è–≥–∞—î –≤ —Ç–æ–º—É, —â–æ —Å–µ—Ä–≤–µ—Ä –º–∞—î —Å–ø–∏—Å–æ–∫ –ø—ñ–¥–∫–ª—é—á–µ–Ω–∏—Ö –∫–ª—ñ—î–Ω—Ç—ñ–≤ —ñ –ø–µ—Ä–µ—Å–∏–ª–∞—î –∫–æ–∂–Ω–µ –æ—Ç—Ä–∏–º–∞–Ω–µ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è –≤—Å—ñ–º —ñ–Ω—à–∏–º –∫–ª—ñ—î–Ω—Ç–∞–º. –ö–æ–∂–µ–Ω –∫–ª—ñ—î–Ω—Ç –ø—Ä–∞—Ü—é—î —É –≤–ª–∞—Å–Ω–æ–º—É –ø–æ—Ç–æ—Ü—ñ, —â–æ –¥–æ–∑–≤–æ–ª—è—î —Å–µ—Ä–≤–µ—Ä—É –ø–∞—Ä–∞–ª–µ–ª—å–Ω–æ –æ–±—Ä–æ–±–ª—è—Ç–∏ –ø–æ–≤—ñ–¥–æ–º–ª–µ–Ω–Ω—è.

–û—Å—å –∫–æ–¥ —Å–µ—Ä–≤–µ—Ä–∞, —è–∫–∏–π –ø—ñ–¥—Ç—Ä–∏–º—É—î –¥–µ–∫—ñ–ª—å–∫–∞ –∫–ª—ñ—î–Ω—Ç—ñ–≤:

```python

clients = []  # –°–ø–∏—Å–æ–∫ –∞–∫—Ç–∏–≤–Ω–∏—Ö –∫–ª—ñ—î–Ω—Ç—ñ–≤

def handle_client(client_socket):

        try:
            message = client_socket.recv(1024)
            if not message:
                break  # –í–∏—Ö—ñ–¥ –ø—Ä–∏ –≤—Ç—Ä–∞—Ç—ñ –∑'—î–¥–Ω–∞–Ω–Ω—è
            broadcast(message, client_socket)
        except:
            break  # –í–∏—Ö—ñ–¥ –ø—Ä–∏ –ø–æ–º–∏–ª—Ü—ñ

    client_socket.close()
    clients.remove(client_socket)  # –í–∏–¥–∞–ª—è—î–º–æ –∫–ª—ñ—î–Ω—Ç–∞ –∑—ñ —Å–ø–∏—Å–∫—É

def broadcast(message, sender_socket):

        if client != sender_socket:
            try:
                client.sendall(message)
            except:
                client.close()
                clients.remove(client)

    print(f"–ü—ñ–¥–∫–ª—é—á–∏–≤—Å—è –Ω–æ–≤–∏–π –∫–ª—ñ—î–Ω—Ç: {client_address}")
    clients.append(client_socket)
    thread = threading.Thread(target=handle_client, args=(client_socket,))
    thread.start()
```
